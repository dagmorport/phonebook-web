# Телефонный справочник — Веб-интерфейс

Корпоративный веб-интерфейс для просмотра контактов сотрудников.

## Описание

Каркас (шаблон) веб-приложения для корпоративного телефонного справочника. Данные сотрудников подставляются из внешнего источника (Active Directory).

## Технологии

- **HTML5** — структура приложения
- **Tailwind CSS** — утилитарные стили, адаптивная вёрстка, тёмная тема
- **Lucide Icons** — SVG-иконки
- **Vanilla JavaScript** — логика приложения

## Возможности

- Поиск по имени, фамилии, должности, отделу
- Алфавитный указатель для быстрой навигации
- Тёмная тема
- Избранное (сохраняется в браузере)
- Копирование email одним кликом
- Адаптивный дизайн (мобильные устройства)

## Использование

1. Откройте `index.html` в браузере
2. Подставьте данные сотрудников в формате:

```javascript
var employees = [
  { last: 'Фамилия', first: 'Имя', dept: 'slug', deptName: 'Название отдела', role: 'Должность', phone: '+7 (XXX) XXX-XX-XX', email: 'user@company.ru', room: '101' },
];
```

## Структура карточки сотрудника

```javascript
{
  last: 'Иванов',           // Фамилия
  first: 'Алексей',         // Имя
  dept: 'apple',            // Slug отдела (для цвета)
  deptName: 'Яблоко',       // Полное название отдела
  role: 'Директор',         // Должность
  phone: '+7 (495) 123-45-01', // Телефон
  email: 'ivanov@company.ru',  // Email
  room: '101'               // Кабинет
}
```
