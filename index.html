О программе

Телефонный справочник — Демо

Веб-приложение для просмотра контактов сотрудников. Данные загружаются из внешнего источника.

Возможности

  • Поиск по имени, фамилии, должности, отделу
  • Алфавитный указатель для быстрой навигации
  • Тёмная тема
  • Избранное (сохраняется в браузере)
  • Копирование email одним кликом
  • Автоматическая синхронизация с AD

Инструменты и технологии

  • HTML5 — структура приложения, один самодостаточный файл
  • Tailwind CSS — утилитарные стили, адаптивная вёрстка, тёмная тема
  • Lucide Icons — lightweight SVG-иконки (поиск, звезда, телефон и др.)
  • PowerShell 5.1 — скрипт экспорта данных из внешних источников
  • LDAP — протокол доступа к каталогам
  • Vanilla JavaScript — логика приложения без фреймворков: поиск, фильтрация, локальное хранилище, делегирование событий
